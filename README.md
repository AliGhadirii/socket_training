```shell
$ apt install python3
```

# **Usage**
## Step0 : Cloning

First of All Clone the Project : 

```shell
$ git clone https://github.com/AliGhadirii/socket_training.git
```

## Step1 : Run Code
You can simply run this code with Python IDEs like pycharm.
also you can run these in terminal with:
```shell
$ python3 filename.py
```
